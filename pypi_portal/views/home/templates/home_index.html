Flask Large Application Example

PyPI Portal is a small demo app used as an example of a potentially large Flask application with several views and Celery tasks.

This application doesn't do much. It runs a Celery task once a day to get all of the package names, summaries, and versions in PyPI and display them in a paginated table.

You can check that out in the Packages link in the Navbar above. You can find out more about this example in the GitHub page. {% endblock %}
